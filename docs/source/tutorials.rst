
Tutorials
=========

The easiest way to learn to use a new package is usually by studying some examples.
In this section we will present some tutorials in order to give you the understanding needed to apply Fieldosophy on your own datasets.

All tutorials here are given in reStructured text format. You can find the equivalent python script in the "examples/tutorials" directory. The python script of a tutorial will have the same name but with a ".py" file extension instead of ".rst".
